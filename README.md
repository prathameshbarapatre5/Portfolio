# Prathmesh Barapatre Portfolio

This is a Next.js portfolio website recreated based on the CodeBucks27 starter code, customized for Prathmesh Barapatre.

## Technologies Used
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

1. **Install Dependencies:**
   Since the environment could not auto-install dependencies, please run:
   ```bash
   npm install
   ```

2. **Run the Development Server:**
   ```bash
   npm run dev
   ```

3. **Open:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Customization
- **Images:** Please replace the placeholder texts/divs with actual images in `public/images/`.
- **Content:** Update `components/Experience.tsx` and `components/Education.tsx` with more precise dates and descriptions if needed.
